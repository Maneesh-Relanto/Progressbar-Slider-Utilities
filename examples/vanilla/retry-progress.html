RetryProgress

1. Exponential Backoff API Call

Simulates API calls with exponential backoff (1s → 2s → 4s). Demonstrates automatic retry with increasing delays.

2. Linear Retry with Timeout

Uses linear backoff (2s → 4s → 6s). Each attempt adds a fixed delay. Perfect for timeout scenarios.

3. Fibonacci Backoff Pattern

Advanced fibonacci sequence delays (1s → 1s → 2s → 3s → 5s). Gradual increase ideal for rate-limited APIs.

4. Manual Retry with Cancel

Fixed 3s delay with manual retry and cancel controls. User can manually trigger retry or cancel operation.
